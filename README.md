# Ponce Garden Maintenance Website

## Project Overview

This repository contains the code for the Ponce Garden Maintenance website, a simple static website designed for a gardening business located in the Bay Area of California. The site aims to provide information about the business, showcase services offered, and enable customers to easily get in touch.

## Features

### Responsive Design

The website is built with a responsive design, ensuring it looks great on all devices, including desktops, tablets, and smartphones. This is achieved using CSS media queries to adjust the layout dynamically based on the screen size.

### Light JavaScript Functionality

We have implemented light JavaScript to enhance user interaction and functionality. This includes interactive elements and smooth navigation features to improve the overall user experience.

### PHP for Contact Form

To facilitate easy communication between clients and the business, the website includes a contact form that uses PHP to send messages directly to the company's email. This ensures that inquiries are received promptly and handled efficiently.

## Technologies Used

- **HTML5**: For structuring the content of the website.
- **CSS3**: For styling the website and ensuring it is responsive across different devices.
- **JavaScript**: For adding interactive elements and enhancing the user experience.
- **PHP**: For backend processing of the contact form to send emails.
